# Wellness London — Project Context

## Product Goal
Build a premium SEO-driven wellness directory for London focused on:
- Saunas
- Cold plunges
- Cryotherapy
- Recovery studios

## Objective
Create a high-quality discovery experience that ranks on Google and converts users into venue visits.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Architecture
- src/app → pages
- src/data → structured facility data
- src/components → reusable UI components

## SEO Principles
- No thin pages
- Every page must include:
  - Intro
  - Listings
  - Context
  - Guidance
  - FAQ
- Start with 5–10 high-quality pages only
- Prioritise intent over scale

## Design Principles
- Premium, calm, editorial feel
- Inspired by Aesop / Soho House
- Clean typography
- Strong spacing
- High-quality imagery (to be added)

## Avoid
- Over-engineering
- Generic templates
- Scaling too early
- Low-quality listings
